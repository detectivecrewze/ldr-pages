## ğŸš€ How to Use

### Viewing the Experience
1. Open `index.html` in a browser
2. Enter the password in the login dialog (default: "forever")
3. Click "OK" or press Enter
4. Navigate through pages using arrow buttons or navigation dots

1. Open `admin/index.html`
2. Fill in the 6-step wizard:
   - Welcome
   - Login Setup
   - Locations
   - Journey Map
   - Shared World
   - The Promise
3. Click "Finish" to download your `data.js`
4. Replace the existing `data.js` with your customized version

## ğŸ¨ Windows 98 Design Elements

### Colors Used
```css
--win-teal: #008080;          /* Desktop background */
--win-blue: #000080;          /* Title bar start */
--win-blue-light: #1084d0;    /* Title bar end */
--win-gray: #c0c0c0;          /* Window background */
```

### UI Components
- **3D Borders**: Using `border-style: inset/outset`
- **Title Bar**: Gradient from navy to teal blue
- **Buttons**: Raised with shadows, depressed when clicked
- **Text Boxes**: Sunken with white background
- **Progress Bars**: Classic blue segmented fill
- **Taskbar**: Gray with Start button

### Fonts
- **Pixelify Sans**: MS Sans Serif replacement
- **VT323**: Monospace for numbers/stats
